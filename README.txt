You must take all libraries from a folder "libries" to "C:\Program Files\Arduino\libraries" and connect all companents:

arduino A5 - display SCL
arduino A4 - display SDA
arduino 5V - display VCC
arduino GND - display GND
button right - arduino GND
button right - arduino D9
button left - arduino GND
button left - arduino D10

Compile code to arduino and have fun!
